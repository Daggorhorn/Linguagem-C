# 🧠 Estruturas de Dados em C

## 📌 Sobre o repositório
Este repositório contém implementações práticas de **estruturas de dados clássicas** utilizando a linguagem C, com foco em aprendizado, desempenho e organização de dados.

O objetivo é aplicar conceitos fundamentais da computação na prática, desenvolvendo soluções eficientes para manipulação e armazenamento de informações.

---

## 🚀 Estruturas implementadas

- 🌳 Árvores Binárias
- #️⃣ Tabelas Hash
- 📚 Pilhas (Stack)
- 🔄 Filas (Queue)
- 🔁 Filas Circulares

---

## 🧠 Conceitos abordados

- Organização e acesso eficiente a dados
- Complexidade de tempo e espaço
- Alocação dinâmica de memória
- Recursividade
- Manipulação de ponteiros

---

## 🛠️ Tecnologias
- Linguagem C
- GCC (compilador)
